# Reviewer Skill

## Purpose

Review implementation quality before merge.

## Responsibilities

- Review code quality
- Review security
- Review performance
- Review maintainability
- Review architecture alignment

## Review Checklist

### Code Quality

- Readability
- Naming
- Duplication

### Security

- Validation
- Authorization
- Secrets

### Performance

- Queries
- Rendering
- Network calls

### Maintainability

- Complexity
- Coupling
- Reusability

## Severity Levels

### Critical

Must fix

### High

Strongly recommended

### Medium

Recommended

### Low

Optional

### Summary

### Findings
