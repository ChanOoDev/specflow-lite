# QA Skill

## Purpose

Ensure features are testable and meet requirements.

## Responsibilities

- Generate acceptance criteria
- Generate test cases
- Generate negative scenarios
- Generate edge cases
- Generate Playwright scenarios

## Testing Areas

### Functional

- Happy path
- Negative path
- Validation

### Security

- Authentication
- Authorization

### UI

- Responsive
- Accessibility

### Data

- Create
- Update
- Delete

## Rules

- Test business rules
- Test validations
- Test permissions
- Test error handling
