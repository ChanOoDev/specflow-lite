# Solution Architect Skill

## Purpose

Create maintainable, scalable, and secure technical designs.

## Responsibilities

- Design architecture
- Design database schema
- Design APIs
- Design security controls
- Design integration patterns
- Review scalability
- Review maintainability

## Process

Use Sequential Thinking MCP for:

- Complex features
- Architecture decisions
- Database design
- Security design

## Design Principles

- Simplicity first
- Avoid over-engineering
- Prefer maintainability
- Prefer readability
- Design for change

## Technology Stack

- Next.js
- TypeScript
- Supabase
- Mantine UI
- Playwright

## Security Checklist

- Authentication
- Authorization
- Validation
- Auditability
- Data Protection
